3  第二章 数据来源、变量构造与样本筛选

3.0.1 2.1 数据来源

本文使用 CSMAR 数据库中的上市公司财务报表、股权性质、行业分类、交易状态和货币供应量数据。为了与题目要求保持一致，样本区间设定为 2010-2025 年，全部变量均基于年报年末值构造。

数据构造阶段使用合并报表，并通过交易状态字段识别异常上市状态。由于成长率变量需要上一年度总资产作为滞后项，因此在前处理环节额外保留了 2009 年年末总资产作为计算缓冲，但最终分析样本仍严格限制在 2010-2025 年。

在具体处理上，本文先将资产负债表、利润表、现金流量表、股权性质、行业分类、上市状态和宏观货币变量逐表标准化，再以证券代码和年份为键进行年度合并。对所有财务指标均统一为公司年度层面，并保证每个公司每年至多保留一条观测记录。这样处理可以避免混入季报、中报或报表口径不一致的观测，从而使后续比较建立在可比的年度信息之上。

3.0.2 2.2 变量构造

核心变量定义如下：

  • Lev = 总负债 / 总资产
  • NPR = 净利润 / 总资产
  • Size = ln(总资产)
  • Tang = 固定资产净额 / 总资产
  • Growth = (总资产_t - 总资产_{t-1}) / 总资产_{t-1}
  • NDTS = (固定资产折旧 + 无形资产摊销 + 长期待摊费用摊销) / 总资产
  • SOE = 1 表示国有企业，SOE = 0 表示非国有企业

其中，Lev 作为因变量，反映企业资本结构中债务融资占总资产的比重；NPR 用于衡量盈利能力，是本文最核心的解释变量；Size、Tang、Growth 与 NDTS 则构成标准的资本结构控制变量组合。根据题意，Liq 在数据中也被保留，但主回归模型并未将其纳入核心控制变量集合。

为了减弱极端值对估计结果的影响，本文对 Lev、NPR、Tang、Growth、NDTS 在年度截面内实施双侧 1% Winsorize 处理。该处理既遵循作业要求，也能够避免个别异常财务指标对回归斜率产生过强影响。

从描述性统计看，样本企业整体杠杆率处于中等偏上水平。全样本 Lev 的均值约为 0.406、中位数约为 0.399，说明上市公司平均而言约有四成资产对应负债融资。与此同时，NPR 的均值和中位数都在 0.038 左右，表明多数企业样本期内保持了正的平均盈利能力。

进一步比较不同产权类型的描述统计，可以发现 SOE 与非 SOE 在多个维度上呈现系统性差异：国有企业往往规模更大、杠杆率更高，而非国有企业通常具有更高的盈利能力和更强的成长性。正因如此，如果不在模型中显式区分产权性质，就可能把产权结构差异混同为盈利能力本身的影响，这也是后续异质性分析不可或缺的原因。
